# 📊 Support Ticket Classification and Prioritization System (NLP)

An end-to-end Natural Language Processing project that classifies customer support tickets into categories, predicts priority levels, and helps in efficient ticket routing and resolution.

---

## 🚀 Project Overview

In real-world customer support systems, thousands of tickets are generated daily. Manually handling them is inefficient.

This project automates:
- Ticket classification
- Priority prediction
- Ticket analysis using NLP techniques

---

## 📁 Dataset

The dataset contains customer support tickets with features such as:
- Ticket ID
- Customer details
- Product purchased
- Ticket subject & description
- Ticket type
- Organization & location

---

## 🧠 Key Features

✔ Text cleaning & preprocessing  
✔ NLP-based feature extraction  
✔ Ticket classification (e.g., Billing, Cancellation, Refund)  
✔ Priority prediction (High / Medium / Low)  
✔ Data analysis and insights  

---

## ⚙️ Tech Stack

- Python 🐍  
- Pandas & NumPy  
- Scikit-learn  
- NLTK  
- Jupyter Notebook  

---

## 🔄 Workflow

1. Data Loading  
2. Text Cleaning (lowercasing, stopword removal, lemmatization)  
3. Feature Engineering (TF-IDF / text features)  
4. Model Training  
5. Prediction of ticket category and priority  
6. Evaluation using accuracy, precision, recall  

---

## 📊 Model Used

- Logistic Regression / Naive Bayes (based on implementation)
- TF-IDF Vectorization for text representation

---

## 📈 Results

The model predicts:
- Ticket category with reasonable accuracy
- Priority level for faster resolution
- Helps improve support team efficiency
